## Technical Approach

1. **Promise rejection safety**: Added `.catch(() => {})` to `.then()` chains that lacked rejection handlers. This prevents unhandled Promise rejections from propagating and potentially crashing the render cycle.

2. **Optional chaining**: Added `?.` guard for `res.Data?.User` to safely access nested properties without risking a TypeError when the intermediate object is null.

3. **HTML sanitization**: Implemented a DOM-based sanitizer that removes dangerous HTML elements and attributes while preserving safe HTML formatting (links, formatting). This is preferable to regex-based sanitization which is error-prone.

4. **v-for key binding**: Added `:key` to satisfy Vue's reconciliation algorithm requirement, ensuring correct element reuse and preventing rendering bugs.

5. **Console log removal**: Deleted all `console.log` and `console.group` calls that served only debugging purposes.
